# aero_break_guard

## 目标
当高情绪题材板块（如商业航天、低空经济、军工、卫星互联网）出现以下情况时，自动降低乐观度，避免系统在龙头断板后仍对板块内个股给出过高评分：

- 板块龙头断板
- 连板晋级率明显下降
- 板块炸板率升高
- 高位股批量走弱
- 风险公告、异动公告增多

## 核心规则

### 规则 1：龙头断板惩罚
如果高情绪题材板块最高辨识度龙头断板，则：
- 所有同板块个股默认降温
- 非趋势中军票优先下调为“观望”
- 非低位首板/低位补涨票，不给“强烈看多”

### 规则 2：高位情绪退潮约束
如果市场处于高位接力退潮阶段，则：
- 即使个股均线形态较强，也不得直接给出极度乐观
- 情绪退潮日，趋势股最高只允许“中性偏多”
- 高位股优先强调回撤风险和止损纪律

### 规则 3：风险公告优先
若个股存在以下任一情况：
- 异动公告
- 风险提示公告
- 明显偏离基本面
- 连续大涨后放巨量

则优先下调结论：
- 强烈看多 → 看多
- 看多 → 震荡
- 震荡 → 观望
- 持有 → 观望或减仓

### 规则 4：趋势股与情绪股分开处理
- 趋势中军：允许保留“中性偏多”，但不允许无上限拔高评分
- 情绪龙头：一旦断板，优先转入风险控制模式
- 修复观察票：只能给“持有/观察”，不得直接给“强看多”

### 规则 5：输出一致性检查
分析输出必须满足：
- 文案中的股票名必须与当前分析股票一致
- 股票代码必须一致
- 买点、止损位、止盈位必须与当前股票价格区间匹配
- 若出现其他股票名称或明显错位价位，则判为无效结果，需要重生成

## 输出偏好
在商业航天板块退潮时，优先输出以下风格：
- 先讲风险，再讲机会
- 先讲板块情绪，再讲个股技术
- 先讲是否还能参与，再讲买卖点
- 避免在高位退潮日给出煽动性看多措辞

## 推荐动作映射
- 龙头断板 + 高位退潮：观望 / 减仓 / 卖出
- 趋势股强但板块转弱：持有 / 观察
- 修复票未放量确认：观望
- 低位补涨刚启动：可跟踪，但不追高

## 不允许的输出
以下情况禁止出现：
- 板块退潮日仍给出“极度乐观”
- 龙头断板后仍给出“强烈建议买入”
- 当前股票页面出现其他股票名称
- 买点和止损价明显来自其他股票
